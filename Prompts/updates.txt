<recommendation>{recommendation}</recommendation>
<specific_steps_taken>{specific_steps_taken}</specific_steps_taken>

3. Calculate percentage improvement:
   - Write out the calculation: (reduction_achieved / initial_footprint) * 100
   - Calculate the percentage improvement

4. Prepare additional steps (if applicable):
   - If implementation was partial, list additional steps the user can take based on the specific_recommendation

  }}
